## 📰 Tentang Proyek

Website berita yang dikembangkan menggunakan **React.js** dan **Tailwind CSS** dengan memanfaatkan **API Berita Indonesia** sebagai sumber data utama. Aplikasi ini menampilkan berita terkini dari berbagai portal berita nasional dalam antarmuka yang modern, responsif, dan mudah digunakan, sehingga pengguna dapat memperoleh informasi terbaru dengan pengalaman membaca yang nyaman.

### 🔗 API yang Digunakan

> **API Berita Indonesia**
> https://api-berita-indonesia-gules.vercel.app/

API ini menyediakan data berita dari berbagai media nasional yang dapat diakses secara real-time dan diintegrasikan ke dalam aplikasi melalui REST API.
